# My Personal Neovim Configuration
This is my personal Neovim setup, using [lazy.nvim](https://github.com/folke/lazy.nvim) as the plugin manager. It includes essential plugins for productivity, code navigation, language support, and a good development experience.

Licensed under the [MIT License](./LICENSE) — feel free to use, modify, or share it as you wish 😉

## Included Plugins & Features

* **lazy.nvim** – Efficient, modular plugin manager.
* **neo-tree** – Elegant file explorer.
* **bufferline** – Tab-like buffer explorer.

> You can easily add or remove languages by editing the [`languages.lua`](./lua/configs/languages.lua) file.
