# My Personal Neovim Configuration

This is my personal Neovim setup using [lazy.nvim](https://github.com/folke/lazy.nvim) as the plugin manager. It includes essential plugins for productivity, code navigation, language support, and an overall enhanced development experience.

Licensed under the [MIT License](./LICENSE) — feel free to use, modify, or share it as you wish! 😉

## Included Plugins & Features

- **lazy.nvim** – Efficient, modular plugin manager
- **neo-tree** – Elegant file explorer
- **bufferline** – Tab-like buffer explorer
- **treesitter** – Parser generator tool for better syntax highlighting and code understanding
- **Mason** – LSP and external tool installer
- **nvim-cmp** – Autocomplete plugin
